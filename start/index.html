# 环境搭建

WARNING

node.js 版本不低于 8.0

# 全局安装 vuepress

npm i -g vuepress
// 或 npm i -g vuepress --registry=https://registry.npm.taobao.org
1
2

# 创建项目

# 新建目录

mkdir vuepress-demo
cd vuepress-demo
1
2

# 初始化

npm init -y
1

# 添加脚本

此操作非必须，添加脚本的目的是为了简化命令。 打开package.json文件，修改scripts

{
  "scripts": {
    "dev": "vuepress dev docs",
    "build": "vuepress build docs"
  }
}
1
2
3
4
5
6

后续可运行命令：

  • 开发：npm run dev
  • 发布：npm run build
